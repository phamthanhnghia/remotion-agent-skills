# Remotion General Workflow

Build complex, custom, or long-form narrative React video compositions using the full suite of Remotion capabilities.

## Contract & Prerequisites

- **Input:** Custom brief, complex multi-scene storyboards, or unique creative requirements.
- **Output:** Fully customized Remotion composition with bespoke component architecture.
- **Triggers:** "custom video composition", "complex multi-scene video", "long form video render".

## Execution Steps

1. Establish technical contract via `/remotion-core`.
2. Define scene timeline structure and props schemas.
3. Import motion patterns from `/remotion-animation` and typography from `/remotion-creative`.
4. Validate with Studio and render with `/remotion-cli`.
